I needed to do something similar but I ran into the same problem (only the first row shows up inside the email). My solution was to attach the multiple rows as a report to the email.

 

Inside the tasks associated with the schedule, I set a burst condition on email address from my report query and then had the "process and save" and "send email" actions indented underneath "execute". The result is an individual email with an attachment of the bursted file.

 

Hope this helps!

This is a slightly old post, but recently we have worked around the single row return by concatenating rows delimited by chr(13)s. It is not an ideal method, but works. Below is an example of our treatment (a scheduled email has %% ArgosData.strTxt %% in it, the result is is multi-rowish output. While you may not need the carriage returns, you may be able to construct the HTML in a similar way- by concatenation into a single row.

 

  

/* Each exception type would return a one-liner with some count or something as a contributing query in a concatenation delmited by ASCII 13 chars */

select

-- *************************************** Begin exception type sentence ********************************************
(
select 'There are '||
(
select count(AS_STUDENT_DATA.ID)
  from BANINST1.AS_STUDENT_DATA AS_STUDENT_DATA,
       BANINST1.SGVCCUR SGVCCUR
 where ( AS_STUDENT_DATA.PIDM_KEY = SGVCCUR.SGVCCUR_PIDM
         and AS_STUDENT_DATA.TERM_CODE_KEY = SGVCCUR.SGVCCUR_TERM_CODE )
   and ( AS_STUDENT_DATA.ENROLLED_IND = 'Y'
         and AS_STUDENT_DATA.REGISTERED_IND = 'Y'
         and AS_STUDENT_DATA.TERM_CODE_KEY = (select STVTERM_CODE from STVTERM
         where sysdate between STVTERM_START_DATE and STVTERM_END_DATE
          and STVTERM_CODE not like 'A%')
         and SGVCCUR.SGVCCUR_PRIORITY_NO = 11
         and SGVCCUR.SGVCCUR_ORDER >0
         and SGVCCUR.SGVCCUR_STDN_TERM_CODE_EFF = (select max(SGVCCUR1.SGVCCUR_STDN_TERM_CODE_EFF) from SGVCCUR SGVCCUR1 where SGVCCUR1.SGVCCUR_PIDM=AS_STUDENT_DATA.PIDM_KEY and SGVCCUR1.SGVCCUR_STDN_TERM_CODE_EFF<=AS_STUDENT_DATA.TERM_CODE_KEY) )
)
||' students with active CCUR records that have priority number 11 in them.' from sys.dual)
-- **********************************************End of exception****************************************************


||chr(13)||  -- Each sentence is concatenated with a chr(13) [carriage return]


-- *************************************** Begin exception type sentence ********************************************
(
select 'There are '||
(
select count(AS_CATALOG_SCHEDULE.CRN_KEY) "RecCount"
  from BANINST1.AS_CATALOG_SCHEDULE AS_CATALOG_SCHEDULE
 where AS_CATALOG_SCHEDULE.TERM_CODE_KEY = (select STVTERM_CODE from STVTERM
       where sysdate between STVTERM_START_DATE and STVTERM_END_DATE
        and STVTERM_CODE not like 'A%')
       and AS_CATALOG_SCHEDULE.DEPT_CODE is null
)
||' courses with null department codes.' from sys.dual)
-- **********************************************End of exception****************************************************


||chr(13)||  -- Each sentence is concatenated with a chr(13) [carriage return]


-- *************************************** Begin exception type sentence ********************************************
(
select 'There are '||
(
select count(AS_STUDENT_DATA.ID)
  from BANINST1.AS_STUDENT_DATA AS_STUDENT_DATA
 where AS_STUDENT_DATA.TERM_CODE_KEY = (select STVTERM_CODE from STVTERM
       where sysdate between STVTERM_START_DATE and STVTERM_END_DATE
        and STVTERM_CODE not like 'A%')
       and AS_STUDENT_DATA.ENROLLED_IND = 'Y'
       and AS_STUDENT_DATA.REGISTERED_IND = 'Y'
       and AS_STUDENT_DATA.CLAS_CODE in (11,12,13,14)
       and (select count(AS_ADVISOR1.ADVISOR_ID) from AS_ADVISOR AS_ADVISOR1 where AS_ADVISOR1.PERSON_UID=AS_STUDENT_DATA.PIDM_KEY and AS_ADVISOR1.ACADEMIC_PERIOD_START <=AS_STUDENT_DATA.TERM_CODE_KEY
       and AS_ADVISOR1.ACADEMIC_PERIOD_END >=AS_STUDENT_DATA.TERM_CODE_KEY and AS_ADVISOR1.ADVISOR_TYPE like 'MAJ%') = 0
)
||' undergraduates with no MAJR advisors assigned.' from sys.dual)
-- **********************************************End of exception****************************************************


||chr(13)||  -- Each sentence is concatenated with a chr(13) [carriage return]


-- *************************************** Begin exception type sentence ********************************************
(
select 'There are '||
(
select count(AS_STUDENT_DATA.ID)
  from BANINST1.AS_STUDENT_DATA AS_STUDENT_DATA
 where AS_STUDENT_DATA.TERM_CODE_KEY = (select STVTERM_CODE from STVTERM
       where sysdate between STVTERM_START_DATE and STVTERM_END_DATE
        and STVTERM_CODE not like 'A%')
       and AS_STUDENT_DATA.ENROLLED_IND = 'Y'
       and AS_STUDENT_DATA.REGISTERED_IND = 'Y'
       and AS_STUDENT_DATA.CLAS_CODE in (11,12,13,14)
       and (select count(AS_ADVISOR1.ADVISOR_ID) from AS_ADVISOR AS_ADVISOR1 where AS_ADVISOR1.PERSON_UID=AS_STUDENT_DATA.PIDM_KEY and AS_ADVISOR1.ACADEMIC_PERIOD_START <=AS_STUDENT_DATA.TERM_CODE_KEY
       and AS_ADVISOR1.ACADEMIC_PERIOD_END >=AS_STUDENT_DATA.TERM_CODE_KEY and AS_ADVISOR1.ADVISOR_TYPE like 'ACAD') >0
       and (select count(AS_ADVISOR1.ADVISOR_ID) from AS_ADVISOR AS_ADVISOR1 where AS_ADVISOR1.PERSON_UID=AS_STUDENT_DATA.PIDM_KEY and AS_ADVISOR1.ACADEMIC_PERIOD_START <=AS_STUDENT_DATA.TERM_CODE_KEY
       and AS_ADVISOR1.ACADEMIC_PERIOD_END >=AS_STUDENT_DATA.TERM_CODE_KEY and AS_ADVISOR1.ADVISOR_TYPE like 'MAJ%') >0
)
||' undergraduates with both ACAD and MAJR advisors assigned.' from sys.dual)
-- **********************************************End of exception****************************************************

"strTxt" from sys.dual
