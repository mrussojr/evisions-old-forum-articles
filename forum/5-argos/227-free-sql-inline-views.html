I'm attempting to create a dashboard that displays things like the average number of institutions our students have previously attended. I understand that you can do inline views in the free sql editor but every time I attempt to do something similar to the code below I get errors.  It suggests to use the AS function to rename a field but when I do that, (AS "institutions") it tells me that the code is an unidentified variable. 

 

Any ideas on how to achieve this?

Select avg(TH.institutions)from (Select count(distinct(shrtrit.shrtrit_sbgi_code)) institutions, a.SPRIDEN_FIRST_NAME, a.Spriden_Last_Name, a.Spriden_ID

From spriden a, sfrstcr, ssbsect, shrtrcr, shrtrit

Where sfrstcr.SFRSTCR_TERM_CODE = '170100'

and a.SPRIDEN_PIDM = sfrstcr.SFRSTCR_PIDM

and sfrstcr.sfrstcr_CRN = ssbsect.ssbsect_crn

and sfrstcr.sfrstcr_term_code = ssbsect.SSBSECT_TERM_CODE

and a.spriden_change_ind is NULL

and sfrstcr.sfrstcr_rsts_code = 'RE'

and spriden_pidm = shrtrit_pidm

and shrtrcr_trit_seq_no = shrtrit_seq_no

and spriden_pidm = shrtrcr_pidm

Group by a.SPRIDEN_FIRST_NAME, a.Spriden_Last_Name, a.Spriden_ID)TH

Try it without the quote marks: 

Select avg(TH.institutions) as Institutions
from (Select count(distinct(shrtrit.shrtrit_sbgi_code)) institutions, a.SPRIDEN_FIRST_NAME, a.Spriden_Last_Name, a.Spriden_ID
From spriden a, sfrstcr, ssbsect, shrtrcr, shrtrit
Where sfrstcr.SFRSTCR_TERM_CODE = '170100'
and a.SPRIDEN_PIDM = sfrstcr.SFRSTCR_PIDM
and sfrstcr.sfrstcr_CRN = ssbsect.ssbsect_crn
and sfrstcr.sfrstcr_term_code = ssbsect.SSBSECT_TERM_CODE
and a.spriden_change_ind is NULL
and sfrstcr.sfrstcr_rsts_code = 'RE'
and spriden_pidm = shrtrit_pidm
and shrtrcr_trit_seq_no = shrtrit_seq_no
and spriden_pidm = shrtrcr_pidm
Group by a.SPRIDEN_FIRST_NAME, a.Spriden_Last_Name, a.Spriden_ID)TH
