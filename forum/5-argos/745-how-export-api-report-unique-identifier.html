If i'm understanding this right, you'd like to export the Argos report API key.  If this is truly what you want, you can set up a connection to the Argos database according to the 'Reporting against MAPS and Argos Databases' document offered by Evisions.  I have a DataBlock that I use to keep track of our reports with API's which uses this query:
 

select Report.Name,
       ReportApi.UniqueId,
       ReportApi.Username
  from Report inner join ReportApi on Report.Id = ReportApi.Id
 where ReportApi.Enabled = -1

 

Further you can create a report that can be run on a schedule to export this info for you, or you could set up a schedule and report that would insert this information directly into your database table.
