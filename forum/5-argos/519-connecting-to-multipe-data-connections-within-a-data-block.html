Kim,

       You can pull data into a dashboard/form from multiple data sources, though if you want the data combined into one multi-column box it gets a bit more difficult. Every form element can use a connection other than the one you set for the datablock's default. I use the ADO connection that most of the data will come from as the default and then other form elements can use whatever connection they need. (See the attached image that shows this. )

      For banded and extract reports, you can use datasets. Again, the report query would gather the data from the source where most of it comes from and the dataset can then augment that data. Extarcts and datasets are one of my favorite combos in Argos. 

     Finally, if you really need to combine multiple sources into one form element, like a multicolumn, you may have to use a database link to the other data source.

I hope this helps,

Bill

Since no one has taken a stab at this yet:

 

 I've dabbled with multiple data sources with some success.  Our Student ID card system exists in a separate world from Banner, but uses the PIDM as an Alternate ID.  Our Library circulation system a report with student name, id, and ID card ISO (bar code), the latter of which is scanned when a student checks out materials.  This export was previously set up in Banner using linked tables or some such.

 

To port the report to Argos, I set up a datablock to generate a list of students who are currently enrolled - It exports name, student ID, and PIDM  from Banner. 

In the Extract report, I set up a new dataset, and changed its connection to IDMS (Our Id System),  set the master dataset in the drop down to ArgosData,  and wrote the query to pull ISO.  (See IDCARDSS1.png for a screen shot of this).

 

Once saved, the ISO became a usable field in the ArgosData section of the datasets area (IDCARDSS2.png).  From there, I added it to the detail section of the extract report as you would any other piece (IDCARDSS3.png), and got the desired results (Name, ID, ISO for each student).

 

Let me know if this helps!
