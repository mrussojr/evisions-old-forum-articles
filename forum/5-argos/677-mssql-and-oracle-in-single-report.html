I opened a ticket with the helpdesk for the same question.  Here is the response:

 

Unfortunately, there currently is no method to use two different databases into one query in Argos. 

We do have an Enhancement request logged with our Engineering Team for this capability.

It is known internally as AR-756 Ability to join multiple databases in SQL query. 

I am adding your name and email to this request, so that if/when it is implemented in a future build or update- you will be notified.

 

If you are looking to combine or join the information, the only method to accomplish this is to link the databases.  So yes, you are unable to combine the two data sources into one object.

I had opened ticket for this around 10 days  ago.  .  The reply:

Argos will only connect to one database per object. If you would like to retrieve information from multiple databases, I recommend using datasets to query the additional information not in the Report Query. You will need to use a comma delimited Extract report instead of a CSV report. You will still end up with a CSV file but Extract reports give you more control. It will allow you to add multiple Detail sections. The report will contain one Detail section for ArgosData (Report Query) and an additional Detail section for each additional dataset.
