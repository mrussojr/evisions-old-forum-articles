Anyone know a way I could write this? I'm wanting to have a conditional where you can pick a certain code or all codes. But even if you want all codes, I still only want it to select all from a certain set of codes? select DUAL.DUMMY,

What I typically do is to create a listbox with the codes I want to display.  But, any query that looks at this listbox also has to replicate the same logic if you want the "all codes" type of functionality.  (I like having the listbox autoselect the first entry if users typically select "all codes").  Here is what I might put in my listbox: 

select STVTESC.STVTESC_CODE as "Code",
       STVTESC.STVTESC_CODE || ': ' || STVTESC.STVTESC_DESC as "Description"
  from SATURN.STVTESC STVTESC
 where STVTESC.STVTESC_CODE IN ('ACLM', 'ACSS', 'ACTE', 'ACTM')
union
select '(All)',
       '(All)'
  from SYS.DUAL DUAL
 order by 1

Then, any query which wants to use the listbox looks like this.  (I picked to query from STVTESC in this example so that it is easier to verify that my code works). 

select STVTESC.STVTESC_CODE,
       STVTESC.STVTESC_DESC
  from SATURN.STVTESC STVTESC
 where STVTESC.STVTESC_CODE = :LB_TestCodes.Code
       or ( :LB_TestCodes.Code = '(All)'
         and STVTESC.STVTESC_CODE IN ('ACLM', 'ACSS', 'ACTE', 'ACTM') )
 order by STVTESC.STVTESC_CODE

 

Now, if I have multiple queries that need to use the selected listbox values, I will create a separate SQL variable which looks like the above query and reference the SQL variable everywhere else.

 

Hope that helps!
