We were getting 'Error Code 404 The system can not find the specified file: 'undefined'' on select reports when using the Web Viewer.  These reports ran fine using the Client.   After a tremendous amount of time and energy we found that the problem was the name of the report itself.   If the report had a # in its name then Web View would err.  We had datablocks with names like 'Pell #1', 'Pell #2'.   The corresponding report names were 'Pell #1 report', 'Pell #2 report'.     Removing # from the report name resolved the issue.
