Hello Jorge,

 

Do you mean the API key is changed? Every time the report is moved or copied, it turns off the API and/or changes the key. I check to make sure API option is checked on the reports and I update the API keys in my application. If the API option is checked, and the API key is correct, but you can't connect, then make sure you're updating the API call when moving to another server or instance. For example, we call the Argos API using CURL in PHP. If we move the Argos instance, we need to make sure the URL in the curl request is updated to the correct address.

 

-Luis
