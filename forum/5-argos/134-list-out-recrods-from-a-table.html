Looks like a cool idea.  Does that query work outside of Argos?  Since FTRPRGA_ACTIVITY_DATE is a DATE format, you probably have to pass in a date value.  E.g., trunc(ftrprga_activity_date) = TO_DATE('01/27/2017', 'MM/DD/YYYY') or trunc(ftrprga_activity_date) = trunc(sysdate).

 

Also, Argos generally does not like using SELECT *; it wants columns to be listed out. I suppose it depends on how you are executing the query.
