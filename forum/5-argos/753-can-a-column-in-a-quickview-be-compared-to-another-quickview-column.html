You could do that.  It sounds like they would be two separate SQL queries.  So, you could write the queries so that the second query excludes the result of the first.   E.g., 

-- First Query
SELECT STUDENT_ID, FIRST_NAME, LAST_NAME
FROM TableName
WHERE FirstQueryCriteria

-- Second Query
SELECT STUDENT_ID, FIRST_NAME, LAST_NAME
FROM TableName
WHERE STUDENT_ID NOT IN (SELECT STUDENT_ID FROM TableName WHERE FirstQueryCriteria)
