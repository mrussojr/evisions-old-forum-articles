I have a list of employees, with gender and their respective employee classes. I'm trying to create a simple banded report that will tell me how the following:

 

Number of female employees: 250

Number of male employees: 254

Number of employee class 1: 215

Number of employee class 2: 214

 

The way I was approaching this was to somehow do a calculation that will count the number of 'F' in a field called 'gender', and then to count the 'm'. Then count each one of the individual employee classes.

 

Is this even remotely possible to do in a single Argos report? 

 

Any help would be deeply appreciated.

Hi!  Yes, that is very possible!

 

In your banded report, add a summary band (if you do not have one already).  In that summary band, add an expression which looks like this: 

SUM(IF(ArgosData.gender = 'F', 1, 0))

This will give you a count of everyone with "F" as the listed gender.  You should be able to extrapolate to get other fields.

 

Hope that helps!
