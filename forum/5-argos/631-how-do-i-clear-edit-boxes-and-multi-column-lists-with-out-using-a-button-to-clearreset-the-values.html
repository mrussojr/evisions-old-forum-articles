Can you be more specific in what you are attempting to accomplish? I think the only other control that has the onClick property is the image control, but you may be objecting to the user having to click anything, not just a button.

 

You can use PL/SQL behind a SQL variable to set a value of edit box

i.e. 

begin

:text_box_name := ''; 

end;
