There may be several ways to accomplish this, but the approach I would start with is to have a SQL variable that checks the user's college. (hopefully you can dig into that assignment by the budget code in their payroll data linked back to the finance account information.)  You would need one for each college with a yes or no result if the user belongs to that college.  

 

The script could basically be:

 

SELECT
    CASE
        WHEN (insert script to select user's college) = 'YU'
            THEN 'yes'
       ELSE null
     END college_check
FROM DUAL

 

Add each logo to the form (place them side by side while you set it up - then stack them when you have them working.)

For each logo go to the properties pane and under "Visible" options select the SQL query for that college.

 

If the results of the SQL is not null.the logo will display for the proper college.
