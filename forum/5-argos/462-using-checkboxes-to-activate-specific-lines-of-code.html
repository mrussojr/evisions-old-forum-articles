Hello,

 

I am creating a datablock and reports for the upcoming spring graduation - and one of the repors that I am creating returns all students with a specific type of hold indicator. My datablock returns all students graduating, however, I want to have a checkbox on my dashboard that allows me to only pull students with said hold if the user desires. My datablock is coded through free type - and I want to add in the code (see below for code) that would only pull back students with said hold indicator. My question is - is there a way to use a checkbox on my dashboard to activate those additional lines of code only when the checkbox is checked?

 

Code I need to add to my datablock - 

 

AND EXISTS (
    SELECT
        'Y'
    FROM
        sprhold,
        stvhldd
    WHERE
        sprhold_pidm = shrdgmr_pidm
    AND sprhold_to_date > TRUNC(SYSDATE)
    AND stvhldd_code = sprhold_hldd_code
    AND stvhldd_trans_hold_ind = 'Y')

 

Thank you,

Ian

Hi Ian,

 

You could create a checkbox on the dashboard. For this example, I'll set the Variable Name to "student_has_trans_hold" and set the Checked Value to "Yes".

 

Then in the DataBlock code, I'd put:

 

AND 
  (
    (
      :student_has_trans_hold = 'Yes'
      AND EXISTS
        (
        SELECT
        'Y'
        
        FROM
        sprhold,
        stvhldd
        
        WHERE
        sprhold_pidm = shrdgmr_pidm
        AND sprhold_to_date > TRUNC(SYSDATE)
        AND stvhldd_code = sprhold_hldd_code
        AND stvhldd_trans_hold_ind = 'Y'
        )
    )

  OR :student_has_trans_hold IS NULL
  )
