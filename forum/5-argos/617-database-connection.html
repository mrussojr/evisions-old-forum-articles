To my knowledge, there is not a way to set the data source at the form level (it is typically set at the datablock level).  However, objects that support SQL (Multi-column lists, list boxes, SQL variables, etc). allow you to pick a different connection at the object level.    You can change it by clicking the "Connection: <default>" link within the SQL Editor. 

 

Note that if you are switching between ODS and Banner, you will likely need to rewrite the query to match the connection source.
