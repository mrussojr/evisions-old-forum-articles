Argos will change any list of values taken from a variable and re-format it

replacing this:

     fieldx = LB_Select_Trm.TermCode

with this:

    fieldx  in (:value1, :value2, :value3...) 

 

I find that my options are to create a variable and add a pl/sql block with a cursor/loop statement that will iterate through my term values and pass them into the procedure one at a time,

or

I can have a global temporary table that will hold the values of my list variable, and then use the temp table to do all of my processing during the current session.

I have not been successful at declaring an array with a list of values taken from an argos control.

 

Or, it could be very simple, such as having the variable on the wrong side of the '=' operator!  It's happened...

Candy
