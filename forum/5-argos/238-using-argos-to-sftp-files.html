Agree with @chamblin.  We user Argos to SFTP files to an external servers daily without any issues.  Double-check your connection and the Authorizations for your connection.  Also, make sure that your Host Address does not contain the port...specify the port below in the correct field.  Then double-check your username, password, and root path (begin the root path with a backslash).  Under SFTP Settings, my setup does not require that either of these boxes be checked.

 

Once you have double-checked all of the above items, if you are still having issues, try the same in Filezilla...this will be a good sanity test for the values that you are using.  If you still have issues in Filezilla, some value is incorrect.
