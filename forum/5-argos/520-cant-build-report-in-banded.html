You can try Setting up a Dataset to create the aggregate numbers.  Set the dataset's parent dataset to ArgosData, and set the Major Code = :ArgosData.MajorCode (Or however you have it set up in the Argos Datablock itself).  We routinely do this with Financial Aid reports (List funds by category in the report itself, and do a dataset to aggregate the data and return total Award).

 

If you're interested, I can try to create a datablock for you to show how we do it.
