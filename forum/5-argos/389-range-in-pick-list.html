You can do this easily in a drop down, either by a manual variable or with SQL. You need to have a text column that will be displayed to the user, as well as a min and max value that corresponds to your ranges (see first image). I manually typed the ranges as an example, but you could just have easily done so with SQL with the values are stored in a table. Finally, in the WHERE clause of the report query, list the column you want to use the range and a BETWEEN:

 

SELECT column1, column 2, ...etc

FROM Table

WHERE column1 BETWEEN dropdown.min and dropdown.max

 

(although use the name of your parameter, not the actual words dropdown).

 

Hope this helps!
