I am trying to create a datablock that will allow me to check on user activity by looking backward date-wise.  What I am having trouble doing is using the SQLITE date funciton to compute a date in the past, from a parameter.

 

In other words, I have a parameter that says, "Enter # of days to look back", and want to compute the date as the current date minus the number of days entered.

 

I have tried the following but it does not work:

 

select SysUser.Name,
       SysUser.CreatedDate
  from SysUser
 where strftime('%Y-%m-%d',CreatedDate) > date('now','-:Parm_LB_Days days')

       and :Parm_LB_Days IS NOT NULL
 

If anyone can assist, that would be appreciated.

Try using the julianday function - just make sure to use utc if you are calculating down to the hours since julianday is GMT.   

 

   julianday('now') - julianday(:parm_LB_Days_days,'utc')
