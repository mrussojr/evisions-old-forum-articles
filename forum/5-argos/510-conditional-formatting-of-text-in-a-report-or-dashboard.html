Lisa - 

 

 As far as the Dashboard, you could actually create two text boxen:  one with a red "Yes" and one with a black "No".  Next create a sql variable that returns a value if you want the label displayed and a null if you don't.   Then set the object's visibility to the SQL variable, and it will display the appropriate label.

 

 I made a quick version with a radio button to show you what I mean - take a look at it and let me know if you have any questions!
