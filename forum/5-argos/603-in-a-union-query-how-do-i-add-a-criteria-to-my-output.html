In a datablock that is a UNION query, how do I add a criteria to my output CSV file?  My output csv file, in the "design" / "filters" tab asks for a field name, however, in a union query, the field needs to have delineation from one part of the union to the next part.  In other words, the first query might have FieldA but for the next part of the union, the same field would need to be called FieldAa.  So when you have the same field, but with different names, what needs to be done?

setup your report query to have the union as a nested subquery:

 

SELECT *  
FROM
  ( SELECT 
        SFRSTCR_PIDM AS PIDM,
        SFRSTCR_TERM_CODE AS TERM,
        SUM(SFRSTCR_BILL_HR) AS CREDITS
    FROM 
      SATURN.SFRSTCR
    WHERE
        SFRSTCR_TERM_CODE = '201810'
    GROUP BY
        SFRSTCR_PIDM,
        SFRSTCR_TERM_CODE
    UNION
    SELECT
        SFRSTCR_PIDM AS PIDM,
        SFRSTCR_TERM_CODE AS TERM,
        SUM(SFRSTCR_BILL_HR) AS CREDITS
    FROM 
        SATURN.SFRSTCR
    WHERE 
        SFRSTCR_TERM_CODE = '201710'
    GROUP BY
        SFRSTCR_PIDM,
        SFRSTCR_TERM_CODE
  )
WHERE CREDITS > 12
