I don't think you can.  As far as I know, you can only adjust the text size associated with the checkbox. 

 

You could use a different form control, like an edit box, and make it look like a checkbox with a single character "X" representing checked and a single space " " representing unchecked.  Then you could place a "hidden" checkbox control in the same location as the edit box, and as the user clicks the hidden checkbox it places an"X" or " " in the edit box.

 

A simplified example is attached.  It expects an Oracle database and queries a limited number of obscured PIDMs from SPRIDEN.
