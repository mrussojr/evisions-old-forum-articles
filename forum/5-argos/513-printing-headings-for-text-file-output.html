WelchK - 

 

You should be able to do it with the CSV Export.  There's a "Write Column Header" check box that writes the header, and you can change the delimiter to a pipe as well.  If you're just doing a straight dump of the Argos data, I'd explore that route.

 

If you're using multiple datasets or need the control that an extract report offers, the only way I know of is to create a header section and manually create constant fields and set their value to the desired column headers.
