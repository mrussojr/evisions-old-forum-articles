If you mean BlackBoard Transact, I believe that is an Oracle database so it's very similar to setting up a data connection to Banner. You'd have to know the SourceID (SID), HostName, Port, Service Name, plus a username/password for accessing the database. Create an entry with that info in the TNSNAMES.ORA on the MAPS server. Then from the MAPS app on the MAPS server, you'd go into Data Connections and Add a Connection. Configure the Oracle driver with the Data Source = the SourceID/SID and username/password. Once the data connection is working (the Test button doesn't give an error), you then have to authorize Argos to use that connection (that's in the far right column of the Data Connection screen).
