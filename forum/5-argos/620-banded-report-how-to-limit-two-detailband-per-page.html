You could try having a pseudocolumn using ceil(rownum)/2 on your report query.  The column will return 1,1 for first 2 rows, 2,2 for next 2 rows, etc..

 

ex.  select ceil(rownum/2) pagenum, a.* from table a

 

Then have a header band that groups on this pseudocolumn and set the header band to page break before or after.

just add the pseudocolumn to your query

 

say your query is

 

select instructor_id,

              course_id

from instructor,courses

where ...

 

then it becomes

 

select ceil(rownum/2) pagenum,

              instructor_id,

              course_id

from instructor,courses

where ...
