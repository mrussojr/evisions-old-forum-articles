A .CSV file has no knowledge of justification.  I suspect the file is being imported into Excel.  Excell will treat 99999 as a number and right-justify it.  99999-9999 will be treated as text and be left justified.

 

Open up the .CSV to see if quotes are around the field.  If not, they will need to for it to be displayed in Excel as expected.
