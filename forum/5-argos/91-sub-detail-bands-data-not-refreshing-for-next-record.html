For example the first record in the report displays the correct data in the sub detail bands but when I go to the next record the prior records data is displayed in the sub details.

 

I've created these reports before but cannot find the cause of this.

Band Group “Group Header 1” 

Master is Sub Detail below header 1

Force New Page Before

 

    Sub Detail - Connect to a Dataset

    Master - ArgosData

 

Group Footer - Linked with Band Group “Group Header 1” Above

 

Child Band for a spacer

 

Band Group “Group Header 2”

Master is below Sub Detail below header 1

 

    Sub Detail - Connected to a Dataset

    Master - ArgosData
