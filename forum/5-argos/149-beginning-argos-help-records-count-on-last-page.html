Beginning Argos:  Records Count on Last Page

 

I have a Student Hold report to modify.  I am trying to find a way to print a record count at the very end of a banded report.  Currently, my report prints a count at the end of each page.  How can I get this record count to print only on the very last page?

 

Thank you in advance!

Unless I am missing something, you should be able to : 

  • Right-click on your bottom band 'Band3' 
  • A menu pops up, left-click 'Edit'
  • Left-click the box under 'Only print if expression is true:' 
  • Enter the following expression,  PAGENUMBER = PAGECOUNT
