Yes it is. You might be able to get it to work just using the properties for Checked and Unchecked values. The way I have done it is with a sql variable that looks for the checkbox value and then tie the visibility of the form element to that.

Some thing like this:

Select 1 as "Checked" from dual where :CheckBox1 = 1

And then choose the value "Checked" in the Visible property.
