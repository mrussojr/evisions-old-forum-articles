I am very new to Argos and trying to create my first DataBlock.  I'd like to use the Visual Design tool to create my query but finding that it is either very limited or I'm not using the tool correctly. I have a radio button asking if non-matriculated students should be included - yes or no.  If yes, include all majors.  If no, exclude major codes like 'NM%'.   Is it possible to use parenthesis - basically something like this?  

 

and (:Main_rb_NonMatPrompt = 'Yes' or (:Main_rb_NonMatPrompt = 'No' and sgbstdn_majr_code not like 'NM%'))

 

Do you generally use the Visual Design tool to get started and then switch to Free Type to get the correct results?

You can do it one of two ways:

 

1. Add a new group in the where clause(using the blue plus sign at the bottom left) and then it works as if there are parentheses around that group. This is probably going to be more annoying for this case because you would need two groups.

 

2. Add a new group and then click the little sql button in the bottom right. This allows you to type free text that will be grouped by parentheses.
