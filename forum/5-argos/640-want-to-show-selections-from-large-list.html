You can easily write a list box to be equal to the values selected in the first box.  It's easiest if the values from the first box are driven via SQL. For example, if my first list box uses a validation table: 

select stvterm_code,
       stvterm_desc,
       stvterm_code || ' - ' || stvterm_desc as "display"
from   stvterm
 where substr(stvterm_code,5,1) in ('9','2','5','1')
       and stvterm_start_date between sysdate - 825 and sysdate + 60
 order by stvterm_code desc

Then my list box SQL can read  

select stvterm_code
from saturn.stvterm
where stvterm_code = :MultiColumn1.code

Note: Since the first box is multi-select, you'll need to structure your query like the above, and cannot simply use something like  

select :MultiColumn1.code alias
from dual
