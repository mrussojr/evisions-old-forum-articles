Is there a way to create a blank row between rows in an Argos banded report.

I would like the Detail Band to cause a break or blank row between certain rows.

I have an SQL query and it is almost impossible to get the query itself to create a blank between SQL rows.

 

I have a query that produce data in the form of:

 

Title                                 Number

Bussiness                     23

Business                       25

Technology                 33

Public Service           34

Public Service           35

 

I would like my banded report (Detail Band)  to look like the following:

 

Title                                 Number

Bussiness                     23

Business                       25

 

Technology                 33

 

Public Service           34

Public Service           35

Hi,

 

One way to achieve this is with group bands.  Make sure you are sorting by the Title column.  Then, create a Band Group using the Title as the Group-by field.  The Group Header can have a height of zero.  The Group Footer height will be the difference between each group.

 

HTH.
