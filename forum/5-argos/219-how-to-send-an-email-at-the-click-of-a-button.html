The only part I know that you can do is open the default email software on the machine being used. 

in the "On Click" property choose hyperlink and in the event properties type    Mailto: name@domain.edu

 

You can use the Expression Builder to retrieve the user's email from the system variables.  (or just type mailto: %%:$User.EmailAddress%% )

 

Problem is, there isn't a way to have it open Google mail or anything like that to my knowledge if the user is only set up to work with web-based mail.

We use the banner package gokemal.  This should already be installed and operational.

 

Create an Argos variable which will be triggered by a button like this: 

 

DECLARE

lv_email_success VARCHAR2(01);
lv_email_reply   VARCHAR2(1000);
lv_text  varchar2(5000) := 'text to be displayed in the email body';

 

BEGIN

 

if (place your execute button object name here) is null then
   goto endjob;
   
end if;
  
-- send email 
      gokemal.p_send_email(
            p_to_addr   => 'to email address goes here',
            p_from_addr => 'from email address goes here',
            p_subject   => 'email subject goes here',
            p_message   => lv_text,
            p_priority  => 3,
            p_success_out => lv_email_success,
            p_reply_out => lv_email_reply);

<<endjob>>
null;

 

END;
