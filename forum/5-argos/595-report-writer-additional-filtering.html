The filter you wrote should work.  Is it possible that "Certificate" has an uppercase "C"?  Perhaps a filter on a lower case version: 

lower(shrtmcm_comment) LIKE '%certificate%'

If you're asking how to do case-insensitive filters on listboxes and multi-column listboxes, I don't have a good answer there.   What I typically do is give the user an edit box they can optionally fill in.  Then, in my queries, I do something like this: 

WHERE
    (:ED_Search IS NULL
    OR UPPER(column1) LIKE '%'+ UPPER(:ED_Search) + '%'
    OR UPPER(column2) LIKE '%'+ UPPER(:ED_Search) + '%'
    OR UPPER(column3) LIKE '%'+ UPPER(:ED_Search) + '%')

This tends to be more intuitive for our users.  This also means I can search on multiple columns at once.
