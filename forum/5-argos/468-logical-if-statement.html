There is an IF function (similar to Excel) that you can use.  Here is an example I use on my reports: 

'Submission Range:  ' +IF(GETVARIABLE('DE_SUBMIT_START') = '', 'BoT', GETVARIABLE('DE_SUBMIT_START'))+ ' -- ' +IF(GETVARIABLE('DE_SUBMIT_END') = '', 'EoT', GETVARIABLE('DE_SUBMIT_END'))

Basically, I have two date edit boxes, DE_Submit_Start and De_Submit_End.  If they are blank, then either "BoT" or "EoT" (for Beginning/End of Time) is displayed.

 

You can do nested IF statements.  But, I do not have a handy example of that.

We use Oracle and I don't think I can use an IF statement in the datablock designer - visible fields.  Two options would be DECODE or CASE.

DECODE (SPBPERS.SPBPERS_ECTC_CODE,1,'STRING X','STRING Y')

else
