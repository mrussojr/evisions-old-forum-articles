You'd probably have to create a code for that in the appropriate STV-table and then start adding that code to students in Banner. Assuming you had an accurate list of who they are. You might either put the code in the same place you track the regular veteran code (STVVETC/SGRVETN?) or add it as an attribute to the student (STVATTS/SGRSATT).
