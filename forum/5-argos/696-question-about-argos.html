So if anyone can please help me understand the radio buttons that would be greatly appreciated. Here is a snippet of code that I would like to modify in order to display different information when a different radio button is selected:

 

select guruobj_object as "Object",
       GUBOBJS_DESC as "Description",
CASE
    WHEN guruobj_role = 'BAN_DEFAULT_M' THEN 'M'
    WHEN guruobj_role = 'BAN_DEFAULT_Q' THEN 'Q'
    ELSE guruobj_role
END as "Role"
from bansecr.guruobj, GUBOBJS
where guruobj_userid = :UserClass3.CLASS
and GURUOBJ_OBJECT = GUBOBJS_NAME
and GURUOBJ_ROLE in :radOptions.Value
order by guruobj_object
