Hi,

 

I'm trying to convert a web created report to Argos but I'm running into a formatting issue. I'm able to break this report into separate banded reports but I would like to have it as one report.  It's two different queries and so far, I haven't had much luck with this. Here's the breakdown; 

 

[Title Band]

[Group Header group by a column in the ArgosData (Master)]

[Group Header group by another column in the ArgosData (Master)]
[Detail Band]

 

Once the records from this group is completed, it should print the next group based on the query from a Dataset.

 

[Group Header group by column in the DataSet]

[Group Header group by another column in the DataSet]

[Detail Band]

 

It's an evaluation response report.  The first group is based on numerical records (from the main datablock query), while the second group is based on textual responses (dataset query).

The problem is, you can only have one Detail band. You'll need to leverage a subdetail band:

 

With subdetail bands and a dataset inside your report, it would look like this:

[Title Band]

[Group Header group by a column in the ArgosData (Master)]

[Group Header group by another column in the ArgosData (Master)]
[Detail Band]

[Group Header group by column in the DataSet]

[Group Header group by another column in the DataSet]

[SubDetail Band with "master" value set to the Dataset]

 

Option 2 is to restructure the sql of your main report query and combine it with your dataset into one query (likely need a union) so that there is only one detail band.
 

[Title Band]

[Group Header group by a column in the ArgosData (Master)]

[Group Header group by another column in the ArgosData (Master)]
[Group header with columns from the original Detail Band]

[Group Header group by column in the DataSet]

[Group Header group by another column in the DataSet]

[Detail Band]

 

I've done it both ways. The major reason to restructure is if there is a lot of "parent" records since the dataset can cause serious performance issues - it's basically run for every single row of detail data.
