I'm working with our Adaptive Lab staff and some faculty who are visually impaired. We are wondering if other institutions have visually impaired users and what screen readers they use? Are you using Argos Web Viewer? Which version? Which browser? Are there any tips or tricks we should know about from a developer's stand-point or a user's stand-point?

 

We have found that Web Viewer works better with the screen reader of choice here, JAWS, but not so much with Magic. The users are telling me the edit and list boxes on the dashboard are not acting like standard edit and list boxes they encounter on regular web pages. We also tested Argos Developer and found that JAWS didn't read a lot of what is displayed so that won't work.
