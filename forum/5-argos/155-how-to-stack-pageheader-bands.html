I just tried a basic version - two details bands, each with "print if expression is true" set to different values of a column in my data (i..e, print if col 1 = value1 on band1; print if col1 = value2 on band2, both of which are details bands). They didn't programmatically swap based on values of the data like I think you are wanting.

 

I'm not sure you can do what you need with a static image. If the image is stored in your database, you might be able to do it based on what I'm seeing for the element properties but I haven't worked with database images much myself.
However, for your RTF problem, you can "switch" which values you want to display by using a decode or case in your SQL:

 

select....
case when col1 = 'value' then null else col end display_field

from ... etc.
 

And then refer to display_field in your RTF as a variable: %%ArgosData.display_field%%
