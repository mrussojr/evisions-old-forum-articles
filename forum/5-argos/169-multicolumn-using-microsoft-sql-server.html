Hi Joseph - I ran into the same error for a multi-select parameter (Scenario #1 using "=").  As you described, it worked fine with a single selection but threw "Error while executing 'parm_MC_Results'. Unexpected ':'" when multiple values were selected.   What worked for me was to move ALL the parameters in the WHERE clause to the right side of the equal sign - for example:

:parm_MC_DonorCode.Code = '-ALL'

was changed to 

'-ALL' = :parm_MC_DonorCode.Code

 

I've seen a few references in other forum threads about having the parameter on the right side of the operator but haven't seen an explanation as to why yet.
