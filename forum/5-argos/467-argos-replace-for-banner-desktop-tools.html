Since (generally) users cannot see each other user sessions in TOAD, you will need to log in to TOAD and Desktop Tools with the same user account.  They do not have to be on the same machine.  Alternatively, you could log into TOAD with a database admin account.  Related, you could use auditing functionality within Oracle to see what commands are being ran.
