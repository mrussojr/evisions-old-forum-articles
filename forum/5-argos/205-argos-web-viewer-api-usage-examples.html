Not javascript but:

 

I was successfully able to get the Argos Web api up and running within an .NET project I wrote in C#. I googled each piece of the Web API example to find how to do something similar in c#. I think the hardest part was figuring out how to set up the classes properly (for getters and setters) to make the various API calls. The end result was code that executed on page load to call the necessary APIs for authentication/session/token. Those are all returned JSON, so I used a built-in parser to get back the various data components needed to get the next API (So you have to create a session, authenticate, then use the token to actually call the datablock)

 

Once i had all that, I then dumped the results to the empty iframe server control on my page by setting the src attribute.

 

Hope that helps!
