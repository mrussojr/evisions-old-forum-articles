How would you do the following report:

 

Characteristic        Total

Male                              23

Adjuncts                     223

This sounds like you could use the OLAP tool.  It performs like a pivot table but it can do more.  All the SQL for the OLAP is contained within the tool and does not have to include any of the calculations.  You set the dimensions and the measures and let the tool do all the calculating.  I have created a couple OLAPS  for our Budget team and they pull in a ton of data.  Also, different users can set it up the way they want to see the data and save that as a saved setting so they can get their custom view quickly each time it opens.  You create one kinda like you do a dashboard.  It's one of the tools available to add to the page.  The wizard helps with the set-up.

 

Hope this helps.
