Hi Susan,

 

In your Schedule, if you edit the 'Process and Save Task' and select the Options button (currently only available for Banded Reports) you will receive a dialog where you can specify the password of a document.  Here is a link to our in-product help with more information.

 

https://webhelp.evisions.com/HelpFiles/Argos/en/Default.htm#cshid=268775433

 

Under the Advanced Tab, there is a section titled "Encrypt Document".  Each of the provided fields gives you the typical Argos Expression Builder via the ellipsis (...), to which you can build a dynamic password based on the user received the bursted document.

 

Hopefully you find this helpful.
