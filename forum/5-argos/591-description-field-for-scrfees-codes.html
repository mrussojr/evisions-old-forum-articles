The fee code is actually the field named SCRFEES_DETL_CODE at the catalog level and is SSRFEES_DETL_CODE at the section level.  Both are validated against the TAISMGR.TBBDETC table.  That way the students' charges for the fees shows up in the fees owed/paid areas.

 

the field TBBDETC_DESC gives the description.

Some of what I know is from using Banner since 1995 in different jobs where I entered data. 

 

There are views behind a lot of forms and if you have Toad or SQL Developer or whatever, you can see the code behind the view.  It helps to see where the data comes from. Many of the views are in BANINST1

 

I also have a quick query I run to find any tables using fields with similar names:

 

SELECT  DISTINCT OWNER, TABLE_NAME, COLUMN_NAME
FROM ALL_TAB_COLUMNS
WHERE COLUMN_NAME LIKE '%_PTRM_%'   ---change this search string as needed
AND OWNER NOT LIKE 'BANINST1' 
ORDER BY OWNER,TABLE_NAME;
