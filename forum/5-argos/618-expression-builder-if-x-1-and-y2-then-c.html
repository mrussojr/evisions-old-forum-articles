"And" is supported in the Expression Wizard if you adding an expression field in a  banded report.  Using Expression Builder, then you can accomplish the same thing with nested If statements.  What I typically do is to make complex expressions as part of my SQL query and have it return a bit.  Then, it is much easier to use that bit value in Argos.
