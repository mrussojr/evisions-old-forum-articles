Hi Ian,

 

You can do this:  

 (:textbox is NULL or table.value = :textbox)

You would have to repeat the above code for each textbox.  I do this a lot with date edit boxes.  Users can input a start date, and end date, neither, or both.

Text boxes must not be marked as required. It looks like you may be using drop down boxes. For those, I make them required. But also UNION in a dummy record typically with the text of “(All)”.  I also make sure that it’s sorted first and automatically selected. My query then looks for the drop down box to equal “(All)” or the database value.
