The best place to make the file name a dynamic variable is in the Email Task --> Attachment FileName field. 

 

In the Attachment window you can select the %%ProcessFileName%% variable, but to configure the name of the file you will use the ellipse button to the right of the file name field.  Here you will find the expression builder, where you can embed values from your report data or other variables.

As an example, I have a letter that bursts on a particular student, on a particular date, and for a particular term - all of which I want to be reflected in the file name.  Here is what that expression would look like...

 

Letter_%%{ArgosData.FIRST_NAME}%%_%%{ArgosData.TERM_CODE}%%_%%FormatDate(Now(), "yyyymmdd")%%.pdf

 

...which would generate a file named:  Letter_Brian_201519_20181014.pdf for this particular student -- and the file name would change based on the current report record that is bursting.
