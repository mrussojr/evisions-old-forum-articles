This issue has been resolved.  I had tried searching the internet for similar issues and didn't think of simple solutions.

I opened the Execute The Report window.

Opened Task Manager

Found it listed in the applications tab

right-clicked and Maximized it

grabbed the maximized window and moved it

 

Now I can use it!  It was just somehow hidden off-screen.
