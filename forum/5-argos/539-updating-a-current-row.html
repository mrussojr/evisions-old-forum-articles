In your SQL variable that updates, you need to specify a where clause and a condition using a column from the multi-column parameter. For example, I have a datablock and a SQL variable that inserts values into a table based upon the selected items in the multi-column using the crn column: 

begin

insert into (table_name)

(column1, column2, etc.)

select ssbsect_term_code, ssbsect_crn

from saturn.ssbsect

    where ssbsect_term_code = :parm_dd_term
              and ssbsect_crn = :parm_mc_crselist.SSBSECT_CRN;

end;

 

The parm_mc_crselist is my multi-column box, and ssbsect_crn is from the query behind it. Argos is smart enough to know when you use the = in a where clause for a multi-column that is set to multi-select, to grab all values and treat it like an "in" clause automatically.

 

I hope that makes sense and helps!
