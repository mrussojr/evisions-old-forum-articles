So I've accomplished this in some reports, the only hitch is that you have to be ordering by the field you want to count at some point.  You can add a group header to your detail row that groups by your field.  Add an expression, lets call it UNIQUE_COUNTER, that uses the COUNTER function and set it not to print.  Then set the header to have a height of .01.  Finally  in your footer you can create an expression with the function GETEXPR('UNIQUE_COUNTER') which will print out the last value of the UNIQUE_COUNTER.  This may or may not work depending on your existing grouping and ordering.
