Well... I still haven't found a way, but I can get close to what I want by making a memo data aware and using this sql variable:

 

select
to_char(:mc_Insert1.SPRCMNT_CONTACT_DATE,'MM-DD-YYYY')||' by '||upper(:mc_Insert1.SPRCMNT_USER_ID)
||chr(13)||:mc_Insert1.SPRCMNT_TEXT  as Date_Note
from Dual
