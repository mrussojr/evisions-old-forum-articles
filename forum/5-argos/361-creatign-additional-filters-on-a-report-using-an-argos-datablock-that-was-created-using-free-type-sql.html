Yes!  Here's how to do it:

 

in your free type report sql, at the top you'll see an icon that says "Insert Special"

(see Capture1).

 

special phrases will be inserted into the bottom of your query (see Capture2).  If you do not have a where clause, then use "new filter", if you have a where clause, the use "add filter".  You can do the same with the sort order.
