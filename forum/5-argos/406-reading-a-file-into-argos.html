You could also place the file in a folder on the MAPS server (or somewhere MAPS can see it) and create an ADO connection to the file. This would allow for Argos to see it and use it in a datablock. The datablock would just have different parts that connect to the different data sources, Banner and the CSV. Datasets on the report side could also be a possibility.
