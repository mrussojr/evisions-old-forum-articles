There are 2 things to know here.  You need to be at least version 6.0 to resolve the flat line chart problem.   A lot of work was done to improve charting in the later releases.  You'd be best served to move to the latest release.  To see the changes between releases, review the info at:  https://webhelp.evisions.com/releaseguides/argos/default.htm#Version Matrix.htm%3FTocPath%3D_____17 

 

The other concern about charts behaving differently - yes, they are different.  I don't remember which version began the options of chart types.  The client "Classic" chart has a lot more capability to control the chart design.  But much of it won't display in the WebViewer.  The solution at this point is to create a Standard Chart which looks about the same in both Client and WebViewer.  Pie charts are still lousy though.  I rarely use them.  The later versions of Argos display this when you first begin to create your chart:
