Not directly, but you can simulate that effect:   Add a second expression field in the footer.  Set it to not print (though you may want it visible during testing).  Use the following expression to change the font color of the first expression field to black or white depending on the report name: 

SETFONTCOLOR('Expression1', IF((GETVARIABLE('$REPORT.NAME') = 'xyz'), RGBCOLOR(0, 0, 0), RGBCOLOR(255, 255, 255)))

I tried doing something like this, but it did not work as I had hoped: 

IF(GETVARIABLE('$REPORT.NAME') = 'Scratch Report'), SUM(ArgosData.Students), '')

Now, if you don't need anything in the footer, it may be easier to set an expression on the footer band to hide/show the band as needed.

 

HTH

Setting a property of an expression field based on  the multiple conditions  worked.

 

In this example  I am setting Expression2 field color based on Seqnum values.


SETFONTCOLOR('Expression2', IF(((ArgosData.SEQNUM <>'1'  )  and (ArgosData.SEQNUM <>'4' ) and (ArgosData.SEQNUM <>'7' )   ), RGBCOLOR(0, 0, 0), RGBCOLOR(255, 255, 255)))
