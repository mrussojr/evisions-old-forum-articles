Vijaya,

    You can use the Visual Design controls to copy the whole visual design and paste it, see attached image. Then in the form properties where you paste them you just have to set the Is Tab property to Yes. If the all use the same parameter and it is named the same, this should be fairly easy, though you may want to control the individual forms with a button so the datablock doesn't run them all at once, unless you want it to.
