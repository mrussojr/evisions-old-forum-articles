Hi Kevin,

 

Can you do this in the query itself using a case statement when grades in A,B, C then populate the course, else null. And the second column can have the case when grade in F, W then populate the course else null. 

 

Another way to do this is to have a grouping column which would again be a case statement when grades in A,B, C then Pass else Fail. Then in the banded report you can group by this column.

 

I am not sure if that answer your question but let me know if you are trying to achieve something different.
