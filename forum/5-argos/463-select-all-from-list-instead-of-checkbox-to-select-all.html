Hi Ian,

 

In my Drop down DD: I have something like this so they can select All colleges or a specific one from the list. You could also do a multi column if you wanted them to select more than one item.

 

select 'All' "CollegeCode",
       '1 - All' "CollegeDisplay"
  from SYS.DUAL DUAL
union all
select LOV_COLLEGE.VALUE "CollegeCode",
       LOV_COLLEGE.VALUE||' - '||LOV_COLLEGE.VALUE_DESCRIPTION "CollegeDisplay"
  from ODSLOV.LOV_COLLEGE LOV_COLLEGE
 where LOV_COLLEGE.VALUE not in('99')

 order by 2

 
Then in the main query I have an and or.
 
and ( :DD_College.CollegeCode = 'All'
           or UGA_ACAD_STUDY_PROGRAMS_SLOT.COLLEGE1 = :DD_College.CollegeCode)
 
Hope this helps a bit.
