Hi Adam

 

I see you have some questions regarding connecting to your SalesForce data using the new MAPS 6.0 Cloud Connector.  Based on your description, it sounds like you might be attempting to place the actual API link in the Cloud Connector, and that doesn't seem to be working.

 

If you havent already checked out the In-Product help for the Cloud Connector, please check it out using the link below.

http://webhelp.evisions.com/HelpFiles/MAPS/en/Default.htm#Cloud Connector.htm

 

The one thing you will need to confirm:  If the RESTful API link you are using is not a script to collect the data, but the API call itself, you will need to write an HTML/.JS script (or whatever language to your liking) that calls the API link.  The script can be conveniently stored on the MAPS Server under http_files as well.  You can find out more about deploying the scripts under the "Deploy Script Files" header on the help page.

 

Let me know if this helps answer your question!
