Provided you're using a database and version that supports the LISTAGG function, you can create a SQL variable that references your multi-select box.

 

Here is a sample of SQL that references a lit of fund codes and creates a concatenated list separated by commas . . .

 

SELECT LISTAGG(FTVFUND_FUND_CODE, ' ,')

WITHIN GROUP (ORDER BY '') LIST

   FROM FTVFUND

WHERE FTVFUND_FUND_CODE = :main_MC_Fund.CODE

ORDER BY 1
