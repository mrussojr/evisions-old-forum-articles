I have a SQL that I am trying to change from the option of selectin one term to being able to select more than one term.

 

The original SQL that selected one term was:

 

select
    cs.scbcrse_subj_code||cs.scbcrse_crse_numb course_id,
    cs.scbcrse_title c_title,
    cs.scbcrse_credit_hr_low c_min,
    cs.scbcrse_credit_hr_high c_max,
    cs.scbcrse_csta_code c_status,
    cs.scbcrse_activity_date last_updated,
    cs.scbcrse_user_id last_updated_by
from
    scbcrse cs, stvcoll cl
where  cl.stvcoll_desc  = :parm_LB_college.COLLEGE
and   (cs.scbcrse_csta_code                     = :parm_LB_course.code
           or :parm_CB_SelectAllCourses = 'Y' )
and cs.scbcrse_coll_code = cl.stvcoll_code
and (cl.stvcoll_code        =  :parm_LB_college.code    or :parm_CB_SelectAllColleges = 'Y')
and ( cs.scbcrse_eff_term  <= :parm_LB_term.t_code )
order by cl.stvcoll_desc, cs.scbcrse_subj_code||cs.scbcrse_crse_numb

 

Even though I change to a list box instead of drop down, Argos stated "Multiple values exist for variable "parm_LB_term.t_code", however the SQL script has been written to expect only one value for this variable. To accept multiple values for variables only "=" and "<>" can be used for comparison operators. The query used the operator "<="" .

 

In the SCBCRSE table, the active course may not have an effective date of the term selected. It can have an effective date of a term less than that.

Normally it is a simple fix for the error message: you would just change the <= to an = and then set the list box multi-select property to “yes” for the term code parameter piece.

 

However, I do not think that will actually get you the desired result because SCBCRSE is a effective-term table; you’d need to do a max against the term code field to get the correct record (which is how the SQL was originally written) but that does not necessarily make sense when you are selecting multiple terms.

 

Can you explain what the goal of selecting more than one term from this table? Are you attempting to determine “active” courses in the Banner catalogue?
