Eric,

 

I have seen that putting a sort on the query within the datablock slows things down.  If the sort is on the actual report that may help with the time it takes to generate the output.  The other suggestion, if possible, is to put some of the queries within the report and not on the master datablock.   We have used that as well to help shave some time.
