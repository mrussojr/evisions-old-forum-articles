One way to do this would be to create the file using the Extract report type instead of CSV. You first make the extract a delimited file and then add a title band. On the fields tab you add a constant field for each column and enter the column name in the Value field for each column constant. It is a little bit of work if there are a lot of columns, but this is what I do for integration files for external systems.  

Also, I usually add and set up my detail band first and then do my title band, that way I have the detail there to reference when creating the constants. I attached a screenshot to go along with this description.
