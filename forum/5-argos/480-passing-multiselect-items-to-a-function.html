Hi Pablo

 

How about something like this where mc_coll is a multi-column list with multi-select enabled:

 

select s.STVCOLL_DESC
from STVCOLL s
where STVCOLL_CODE = :mc_coll.COLL_CODE

 

I initially tried this, but Argos threw up an error explaining that you must use = or <> when evaluating list results

 

select s.STVCOLL_DESC
from STVCOLL s
where STVCOLL_CODE in(:mc_coll.COLL_CODE)
