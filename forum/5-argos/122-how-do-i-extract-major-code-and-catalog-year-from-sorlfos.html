I am trying to write code similar to what is in Banner INB SHADEGR, to extract the SORLFOS MAJOR CODE and the SORLFOS TERM CODE CTLG and am not having complete luck.  I know I can get the two fields from the SHRDMGR table, but according to what I see in SHADEGR, that information comes from a related SORLFOS table.  I have tried linking up SHRDGMR to SORLCUR in order to get the correct SORLFOS record, but am not getting results I want.

 

If someone can please take a look at my code and make a suggestion, I would appreciate it.

 

Thank you and here is my code snippet, with SORLCUR and SORLFOS :

 

SELECT SHRDGMR_CAMP_CODE                CAMPUS,
                 SHRDGMR.SHRDGMR_PIDM             PIDM,
                 SHRDGMR.SHRDGMR_TERM_CODE_GRAD   GRAD_TERM,
 --              SHRDGMR.SHRDGMR_DEGS_CODE        GRAD_STATUS,      
                 SHRDGMR.SHRDGMR_GRST_CODE        GRAD_STATUS,
                 MAX(SHRDGMR.SHRDGMR_SEQ_NO)      SEQ_NUMBER,
                 SHRDGMR.SHRDGMR_PROGRAM          PROGRAM_GRADUATION,
                 SHRDGMR.SHRDGMR_MAJR_CODE_1      FIELD_OF_STUDY_GRAD,
                 STVMAJR_DESC                     MAJOR_DESC,
                 SHRDGMR.SHRDGMR_TERM_CODE_STUREC CATALOG_YEAR,

                 SORLCUR .SORLCUR_CAMP_CODE       SORLCUR_GRAD_CAMPUS,
                 SORLCUR.SORLCUR_PROGRAM          SORLCUR_PROGRAM,
                 SORLFOS_MAJR_CODE                SORLFOS_MAJOR_CODE,

                 SORLFOS.SORLFOS_TERM_CODE_CTLG   SORLFOS_CATALOG_YEAR,
                 SHRDGMR.SHRDGMR_DEGS_CODE        DEGREE,
                 SORLCUR.SORLCUR_SEQNO            SORLCUR_SEQUENCE_NUM
            FROM SHRDGMR SHRDGMR,
               SORLCUR SORLCUR,
               SORLFOS SORLFOS,
                 STVMAJR STVMAJR,
                 ( SELECT D.SHRDGMR_PIDM, MAX(D.SHRDGMR_SEQ_NO) SHRDGMR_SEQUENCE_MAXNO
                     FROM SHRDGMR D
                    GROUP BY D.SHRDGMR_PIDM
                 ) Z

           WHERE --SHRDGMR.SHRDGMR_TERM_CODE_STUREC   = :PARM_LB_TERM
                 SHRDGMR_CAMP_CODE                = :PARM_LB_CAMPUS
           AND SHRDGMR_CAMP_CODE                = SORLCUR.SORLCUR_CAMP_CODE
           AND SHRDGMR.SHRDGMR_PIDM             = SORLCUR.SORLCUR_PIDM
 --          AND SHRDGMR.SHRDGMR_PIDM = 1125947
           AND SORLCUR.SORLCUR_LMOD_CODE  = 'LEARNER'  -- only learner records
           AND SORLCUR.SORLCUR_CACT_CODE  = 'ACTIVE'     -- only ones that are active
--           AND SORLCUR.SORLCUR_TERM_CODE  <= :PARM_LB_TERM
 
           AND SORLCUR.SORLCUR_TERM_CODE  = (SELECT MAX(B.SORLCUR_TERM_CODE) -- highest term code for active undergraduate learner record
                                                    FROM SORLCUR B
                                                   WHERE B.SORLCUR_PIDM      = SORLCUR.SORLCUR_PIDM
                                                     AND B.SORLCUR_TERM_CODE <= :PARM_LB_TERM
                                                     AND B.SORLCUR_LMOD_CODE = 'LEARNER'
                                                     AND B.SORLCUR_CACT_CODE = 'ACTIVE'
                                                     AND B.SORLCUR_LEVL_CODE = 'UG')
           
           AND SORLCUR.SORLCUR_SEQNO      = (SELECT MAX(C.SORLCUR_SEQNO)
                                                    FROM SORLCUR C -- max sequence for active undergraduate learner record for the term
                                                   WHERE C.SORLCUR_PIDM = SORLCUR.SORLCUR_PIDM
                                                     AND C.SORLCUR_TERM_CODE = SORLCUR.SORLCUR_TERM_CODE
                                                     AND C.SORLCUR_LMOD_CODE = 'LEARNER'
                                                     AND C.SORLCUR_CACT_CODE = 'ACTIVE'
                                                     AND C.SORLCUR_LEVL_CODE = 'UG')  
            
           AND SHRDGMR.SHRDGMR_GRST_CODE        = 'GR'
           AND SHRDGMR.SHRDGMR_PIDM             = Z.SHRDGMR_PIDM
           AND SHRDGMR.SHRDGMR_SEQ_NO           = Z.SHRDGMR_SEQUENCE_MAXNO
           AND SORLFOS.SORLFOS_PIDM             = SORLCUR.SORLCUR_PIDM
           AND SORLFOS.SORLFOS_PIDM             = SHRDGMR.SHRDGMR_PIDM
           AND SORLFOS.SORLFOS_LCUR_SEQNO       = SORLCUR.SORLCUR_SEQNO
           AND SORLFOS.SORLFOS_TERM_CODE        = SORLCUR.SORLCUR_TERM_CODE
           AND SORLFOS.SORLFOS_LFST_CODE        = 'MAJOR'
           AND (SORLFOS.SORLFOS_CSTS_CODE       = 'INPROGRESS' OR SORLFOS.SORLFOS_CSTS_CODE     = 'REMOVED')
           AND SORLFOS.SORLFOS_CACT_CODE        = 'ACTIVE'
           AND SHRDGMR.SHRDGMR_MAJR_CODE_1      = STVMAJR.STVMAJR_CODE

         GROUP BY
               SHRDGMR.SHRDGMR_PIDM,
               SHRDGMR_CAMP_CODE,
               SHRDGMR.SHRDGMR_MAJR_CODE_1,
               SHRDGMR.SHRDGMR_DEGS_CODE,
               SHRDGMR.SHRDGMR_TERM_CODE_GRAD,
               SHRDGMR.SHRDGMR_GRST_CODE,
               SHRDGMR.SHRDGMR_GRAD_DATE,
               SHRDGMR.SHRDGMR_PROGRAM,
               SHRDGMR.SHRDGMR_TERM_CODE_STUREC,
               sorlcur.sorlcur_CAMP_CODE,
               SORLCUR.SORLCUR_PROGRAM,
               SORLFOS_MAJR_CODE,
               STVMAJR_DESC,
               SORLFOS.SORLFOS_TERM_CODE_CTLG,
               SORLCUR.SORLCUR_SEQNO
