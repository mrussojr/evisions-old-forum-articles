Assuming you have a connector setup for the MAPS Data Warehouse, this query should work:

 

select Report.Id as "Report_Id",
       Report.Name as "Report_Name",
       Report.Description as "Report_Description",
       Report.Path as "Report_Path",
       Report.LastRunDate as "Report_LastRunDate"
  from Report
  where datetime(Report.LastRunDate) < (SELECT DATETIME('now', '-365 day'))
  order by Report.LastRunDate desc

You can also search directly in Argos:

  1. Click the Search tab.
  2. Use a percent sign (%) as the search query
  3. Click Advanced Filters to show additional options.  
  4. Check "Reports" and un-check the other options.
  5. In the second (to) Last Run Date, enter the newest date you want to search for. 
  6. Click the Update Results button to search.

I do this to find unused schedules.  Note that the query will take while to run.
