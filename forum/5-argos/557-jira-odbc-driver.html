In MAPS Config, look at the query properties.  Is the Cursor Type set to "Open Forward Only"?  If not, it may need to be set to that.  Also, look at the Cursor Location.  It may need to be set to "Use Client".  I don't have a JIRA connection to look at, so I'm making an educated guess.  Hope this helps!
