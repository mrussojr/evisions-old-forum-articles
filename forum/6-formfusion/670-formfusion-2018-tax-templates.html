Hello Everyone,

 

Currently, tax templates are being updated. They are still expected to be released mid-December and no specific release date has been set at this time. 

 

This is a good time to check the FormFusion version you are currently running. We recommend being on the latest version (version 3.7)  but if this is not feasible, FormFusion version 3.6 is the minimum recommendation. There was an UTF8 issue addressed in version 3.6. See No mapping for the Unicode character exists in the target multi-byte code page for more details.


If you would like to help us test the tax templates prior to the official release, let us know by opening a helpdesk ticket.   
 

Kind Regards,

Kirsten Chung

Hello Everyone,

 

The updates to the 2018 Tax Templates have been completed and they are currently going through the Pre-Release testing.

I want to take this opportunity to thank all of our clients who have volunteered their time for the Pre-Release testing.  

 

If you still like to participate in the Pre-Release testing,  let us know by opening a helpdesk ticket. 

The official release is scheduled for mid-December.  

 

If FormFusion has not been upgrade recently, please note we recommend being on the latest version (version 3.7). If this is not feasible, FormFusion version 3.6 is the minimum recommendation. There was an UTF8 issue addressed in version 3.6. See No mapping for the Unicode character exists in the target multi-byte code page for more details.
