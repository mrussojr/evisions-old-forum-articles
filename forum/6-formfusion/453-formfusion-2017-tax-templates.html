Hi,

 

The Finance patch pcr-000152985_fin8110106, which is the Finance release 8.11.1.6 includes a new version of FAB1099 which is version 8.11.1.6.  The documentation for the 8.11.1.6 release says:

 

DEPENDENCIES: 
        Banner Finance patch pcr-000109245_fin8110104 version 8.11.1.4
        
        Last Version Replacement: 
        8.11.1.6 of fab1099.pc can replace version 8.11.1.4
        
It looks like  you did not have Finance 8.11.1.4 installed.   You're going to want that as it includes a new version of FAM1099 as well as the FAB1099 that's being replaced in 8.11.1.6.  The documentation for the 8.11.1.4 release says:

 

DEPENDENCIES:
                8.11.1.4 of fab1099.pc can replace version 8.9.0.4
                8.11.1.4 of fam1099.pc can replace version 8.10.1.2
 

Once you've installed Finance 8.11.1.4 and then 8.11.1.6 when you run FAB1099 you should see:

 

Username: Connected.

Starting FAB1099 (Release 8.11.1.6)
Run Sequence Number: fab1099 completed successfully

Hi Everyone,

 

Evisions has released a new W-2 template for the 2017 Tax Season in light of the Ellucian patch pcr-000153196_pay8140104 released January 24, 2018.  

 

If you are running pxrac17.pc version 8.14.1 released by Ellucian in December 2017 then you should use the 2017 W-2 Tax Template released by Evisions on December 14, 2017. If you are running pxrw217 version 8.14.1.4 version included in Ellucian's January pcr-000153196_pay8140104 patch then you should use the new 2017 W-2 Tax Template **For pxrw17 version 8.14.1.4** released by Evisions on January 24, 2018. 

 

If you encounter any issues with the mapping or printing of box 15 or 16 in your FormFusion W-2 template, then you may not be using the compatible FormFusion template for your pxrw217 PC version.

 

If you have any questions regarding the Ellucian patch or which process version should be used, please contact the Ellucian Customer Support Center, and refer to defect CR-000153196.  If you have any questions pertaining to the Evisions W-2 Tax Templates, please open an Evisions HelpDesk ticket.
