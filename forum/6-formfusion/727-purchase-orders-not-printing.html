There are lots of possibilities. 

 

First, I presume that you have already started a support ticket with Evisions...

 

Printer setup for FormFusion has to be done either directly on the console of the server, or a MSTSC (MS Terminal Services Client) connection.  Make sure that Windows has the printer defined, and a test page prints. Then run MAPS Client while still on the console, and make sure that the printer is defined, and that the print test page works.

 

Then I would look at the log file from FPAPORD...
