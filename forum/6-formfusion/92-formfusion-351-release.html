Posted December 23, 2016


Evisions is pleased to announce the release of FormFusion version 3.5.1, which resolves the following issue:
 

  • For MAPS servers running Windows Server 2003 or 32-bit Server 2008, the FormFusion mapplet became unresponsive after upgrading to version 3.5.

 
This release is an update to the FormFusion mapplet only.  There are no updates to the FormFusion client or to MAPS.

 

If you have already upgraded to version 3.5 and your MAPS server is not running Windows Server 2003 or the 32-bit version of Server 2008, you do not need to apply this update as there are no changes other than compatibility for these operating system versions. If you have not yet upgraded to version 3.5, you will only see the updated version of the mapplet when you check for updates.
 
As always, we highly recommend installing updates in a test environment before applying them to your production environment. You should make sure that a current backup is available in case of any unforeseen issues.
 
For installation instructions, please refer to the FormFusion 3.5.1 Release Guide.
 
If you have any difficulty with the update, just contact our HelpDesk and we will be glad to assist you.
