I was able to cast them to varchar. e.g.:
 
select DataBlock.Id,
       cast(DataBlock.Name as varchar) as Name,
       cast(DataBlock.Path as varchar) as Path,
       cast(DataBlock.Author as varchar) as Author
  from DataBlock
