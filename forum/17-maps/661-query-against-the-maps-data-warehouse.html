Recall that the database used for the MAPS Data Warehouse is SQLite.  SQLite expects "explicit join notation" like:

 

select ...
from Schedule inner join Report on Schedule.Report_Id = Report.Id
where :frmScheduledReports_btn_Run is not null
 

As a suggestion, build your MAPS Data Warehouse queries using the Visual Designer until you are accustomed to the syntax.  Visual Designer will build the queries using the proper explicit join notation.
