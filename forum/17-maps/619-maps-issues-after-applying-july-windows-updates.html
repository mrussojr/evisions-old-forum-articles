Hi all,

 

We have identified an issue with serveral of the KB updates that have been posted by Microsoft in July 2018.  Applying these updates may severely impact MAPS, preventing it from restarting after the nightly maintenance and backup.  Rebooting the server will temporarily allow MAPS to run again until the next maintenance period.

 

If you are currently experiencing this issue, please read MAPS Service not restarting successfully after recently applied Windows updates for the solution.
