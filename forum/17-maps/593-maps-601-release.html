Posted June 12, 2018

Evisions is pleased to announce the release of MAPS version 6.0.1, which includes the following changes:

 

Enhancements

  • The Cloud Connector can now be used on 32-bit MAPS servers.

Resolved Issues

  • When refreshing data in the Cloud Connector, the fetch operation would sometimes time out and produce a database with empty tables.
  • Resolved an issue in Argos where Unicode special characters were not displaying correctly in the bodies of scheduled emails.

 

As always, we highly recommend installing updates in a test environment before applying them to your production environment.  You should make sure that a current backup is available in case of any unforeseen issues.

 

For installation instructions, please refer to the MAPS 6.0.1 Release Guide.

 

If you have any difficulty with the updates, just contact our HelpDesk and we will be glad to assist you.
