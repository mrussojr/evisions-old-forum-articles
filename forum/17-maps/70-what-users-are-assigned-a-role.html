If by "role" you mean the MAPS User Role, then the simplest way to view the User Role is via the MAPS Config tool found on the product launcher page.

 

If you want to query all usernames for their User Role then the following query would work assuming you have a connector installed for your MAPS Server:

 

--SQLite is used for the MAPS database

select SysUser.Id,
       SysUser.Name,
       case
         when UserType = '0' then 'Report Viewer'
         when UserType = '1' then 'Report Writer'
         when UserType = '2' then 'DataBlock Designer'
         when UserType = '3' then 'Administrator'
         else 'Unknown'
       end as "UserType",
       case
         when deactivated = 0 then 'Active'
         else 'Inactive'
       end as "Status",
       SysUser.Description as "Description",
       SysUser.Guid
  from SysUser
 order by lower(SysUser.Name)
